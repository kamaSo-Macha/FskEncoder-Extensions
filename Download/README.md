# Download section

This folder contains several extensions for the FskEncoder-Application.

After download, please extract the zip file in the FskEncoder application base directory and update the application config according to the hints in the ./bin/xxx.bat and ./cfg/xxx.properties files.

## Disclaimer
This project is still work in progress so the application is as it is! And it is only tested on one single host computer running Win 10.

A lot of work is outstanding like unit tests and javadoc ...
